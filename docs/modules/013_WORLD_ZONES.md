# Module 13 — World & zones expansion

## Scope

- Multiple zones (`zones`, travel gates)
- Zone rules: PvP flag, spawn tables, level range
- Loading screen / transition coordinates
- `dev_static_data` export per zone pack

## Tools

Extend `dev_npcs.php` with zone filter; map metadata in `zones.scene_name`.

## Client

Minimap (optional), zone name HUD, boundary warnings.
